## Installation

```sh
yarn
```
or

```sh
yarn install
```

## Usage

<!-- _For more examples and usage, please refer to the [Wiki][wiki]._ -->

### Development

```sh
yarn dev
```

### Image
You can generate Windows Installer:
```sh
yarn package-win
```

Or Linux:
```sh
yarn package-linux
```
